## Becoming a maintainer

Submit consistent, high-quality contributions over time. After ~5 merged PRs across a few months, an existing maintainer can nominate you. Decision is by lazy consensus of the current maintainers.

## Reaching maintainers

- For project questions, open a [Discussion](https://github.com/d3-inc/agentroot/discussions).
- For Code of Conduct concerns, email **mayank@d3.com**.
- For security issues, see [SECURITY.md](SECURITY.md).
